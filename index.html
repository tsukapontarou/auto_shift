Shift Auto

ローカルブラウザで使うシフト自動作成アプリ

管理者画面でシフト定義と必要人数を調整し、スタッフ入力画面で曜日別の勤務希望を入力します。 現在は同じブラウザ内の localStorage に保存する静的 Web アプリです。

管理者画面へ スタッフ入力画面へ

保存について

このアプリは HTML / CSS / JavaScript だけで動作し、入力内容はサーバーへ送信しません。

  • 現状の保存先は localStorage で、同じブラウザ内だけで共有されます。
  • GitHub Pages 上で公開しても、保存データは GitHub repository やサーバーへ書き込まれません。
  • 別端末・別ブラウザのスタッフとは自動共有されません。
  • ログイン、認証、権限管理、複数端末共有は未対応です。
